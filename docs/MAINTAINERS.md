# Documentation for maintainers

When merging PRs with a merge commit, I like to use the following template for the commit message:

	Merge "<PR title>" (#<PR number>)
	
	"<selected quotation from contributor's cover letter>"

When merging PRs with a squash commit, the default template offered by GitHub is mostly fine, though for it too it's usually better to replace the list of commits in the body of the commit message with selected quotations of the cover letter.
